# Overview
A collection of Subsystems and classes for easily creating data and system heavy games, like RPGs.

## Features
- Gameplay Systems
- Gameplay Modules
- Abilities
- Attributes
- Factions 

## Components
### Core
- Combatant
- Combatnat Extension
- Leveling
- Asset Collection
- Equipment

### Specialized
 - **Input Handler**: An Input action localized to a single actor.
